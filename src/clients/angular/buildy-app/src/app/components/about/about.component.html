About Buildy

What is Buildy?


Want to build a gaming rig, but don't know if your components are compatible ?

With Buildy, you can seamlessly choose between dozens of PC components, check their specifications, and more!

Our software will automatically sort the components and make sure they are compatible with each other. Get the best performance for the money you pay!

  • check_circle_outline   Choose between hundreds of PC Components
  • check_circle_outline   Check their specifications, pricing, and compatibility.
  • check_circle_outline   Don't worry about components that don't go together. Buildy does it for you!

Technology in our software

What's powering Buildy and how do we do it

Java Spring

Java is a general-purpose, multi-paradigm programming language that is the main core of Buildy.

Microsoft SQL Database

Microsoft SQL DB. Simple, yet powerful database solution, housing all PC components! It's a local database.

Material design

Powering the front-end part of the application, and the complete GUI. Very nice, 10/10!
